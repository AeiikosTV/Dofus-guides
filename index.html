Installation

Prêt en 3 étapes

1

Télécharger & extraire

Télécharge le .zip, fais clic droit → "Extraire tout" dans le dossier de ton choix.

→
2

Lancer DofusGuides.exe

Double-clique sur DofusGuides.exe. Au premier lancement, les guides sont téléchargés automatiquement (quelques minutes).

→
3

Naviguer hors-ligne

Le navigateur s'ouvre automatiquement. Consulte tous les guides, même sans internet !

ℹ️ Windows peut afficher un avertissement "SmartScreen" car l'application n'est pas signée numériquement. Clique sur "Informations complémentaires" puis "Exécuter quand même" — c'est normal pour les apps indépendantes.
